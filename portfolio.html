My Portfolio

My First Project: Save the Earth

Save The Earth is a game where you click meteors to get points
There are grey turtles deployed as grey circles that go to random coordinates on the screen.
Each meteor is worth one point and if you score more than 5 points, you get a victory flower.

My Second Project: Survive The AI

Survive the AI is a game where you manuever a newly created
turtle shape away from a computer controlled one using WASD and collecting powerups.
You have to survive a total of 25 seconds to achieve victory.
In addition, the user gets to play this amazing game while listening to amazing background music.

My Second Project: Dino Dash

Dino Dash is a game that is build into a
card for a child in the hospital. This card
is to make them feel better and have a fun game to play.
The user controlls a dino using wasd and has to attempt to get to
the other side. They do this while listening to amazing background music!

Scratch Project: Hungry Dino

This is a dino-themed platformer where you use wasd to get eat chicken.
There is a provided tutorial and multiple backgrounds for the user to learn how to play.
The Dino changes costumes between backgrounds and has a win/lose animation.
